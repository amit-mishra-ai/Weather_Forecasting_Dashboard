# 🌦️ Interactive 7-Day Weather Forecast Dashboard

## 📊 Overview
An interactive **Power BI dashboard** that visualizes **real-time weather data** using a live **Weather API**.  
It provides a complete 7-day weather forecast with insights on **temperature trends**, **humidity**, **wind speed**, **rain probability**, and **air quality index (AQI)** — all updated automatically through API integration.

The goal is to transform raw data into a **visually engaging, easy-to-understand weather story** through Power BI’s powerful visualization and modeling features.

---

## ⚙️ Features
- 🌡 **7-Day Temperature Forecast:** Interactive line chart showing weekly temperature trends.
- 💨 **Live Weather Metrics:** Real-time updates on humidity, wind speed, visibility, UV index, and pressure.
- 🍃 **Air Quality Index (AQI):** Gauge visuals showing PM10, PM2.5, SO₂, NO₂, CO, and O₃ levels.
- 🌧 **Rain Probability:** Dynamic bar chart highlighting daily precipitation chances.
- 🌅 **Sunrise & Sunset Timings:** Automatically refreshed every day.
- 🏙 **Multi-City Comparison:** Compare weather data across **Varanasi, Ballia, Jaipur, and Kolkata**.
- 🎨 **Modern UI:** Dark theme for a sleek, professional, and readable interface.
